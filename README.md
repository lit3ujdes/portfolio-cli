# portfolio-cli

A command-line interface for managing and showcasing your portfolio.

## Features
- View your portfolio items in the terminal
- Add, edit, and delete portfolio entries
- Customize display options for a better look

## Tech Stack
- Node.js for the CLI application

## Usage
1. Clone the repository: `git clone https://github.com/yourusername/portfolio-cli.git`
2. Navigate to the project directory: `cd portfolio-cli`
3. Install dependencies: `npm install`
4. Run the command: `node index.js`
